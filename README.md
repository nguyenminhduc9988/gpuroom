# gpuroom

**Will that LLM actually fit on your GPU?** Zero-install, client-side fit calculator: quant-aware VRAM math, KV-cache/context sliders, max-context finder, best-quant picker, GPU side-by-side compare, and fine-tuning (full/LoRA/QLoRA) memory estimates.
